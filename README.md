Requirements
------------
As stated in Codingame FAQ,we will run under *Python 3.7.2* with : 
 * NumPy 1.16.2
 * pandas 0.24.2
 * SciPy 1.2.1

*Memory limit*: 768MB

Build
-----
Launch __buildcode.py to generate codefinal.py
Use CGSync to upload codefinal.py to IDE
